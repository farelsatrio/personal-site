# Portfolio Website

This project is a personal portfolio built with Next.js, TypeScript, Tailwind CSS, and next-intl for bilingual content.

## Scripts

- `npm run dev` — start local development server
- `npm run build` — create production build
- `npm run start` — run production server
- `npm run lint` — run ESLint checks

## Project Notes

- The site is optimized for a single-page portfolio layout.
- Locale support is available for English and Indonesian.
- Static assets such as profile image and resume are stored in the `public` folder.

## Production Checklist

- Ensure the required image and resume files exist in `public/`.
- Set deployment environment variables if needed for any external integrations.
- Run `npm run build` before deployment.
